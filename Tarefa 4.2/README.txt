TAREFA 4.2 - Adicionando funcionalidade ao formulário [HTML, CSS e JS]

Tendo como base o formulário feito na ultima tarefa, execute os seguintes passos.

1 - Capture todos os dados preenchidos no formulário. Lembre-se de criar uma função que será chamada ao clicar no botão "Enviar". 

2 - Salve os dados em formato de objeto no Local Storage do navegador.

3 - Ao recarregar o formulário, o mesmo deve ser preenchido "automaticamente" com os dados salvos anteriormente no Local Storage.
